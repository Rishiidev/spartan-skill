# Social Preview Upload

Your social preview image is at `assets/social-preview.svg`.

To set it on GitHub:
1. Go to https://github.com/Rishiidev/spartan-skill/settings
2. Scroll to "Social preview"
3. Click Edit → upload `assets/social-preview.svg` (or convert to PNG first with any image tool)
4. Click Save

Size: 1280×640px ✓
